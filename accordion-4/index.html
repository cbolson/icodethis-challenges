The Accordion

The accordion is a graphical control element comprising of a vertically stacked list of items, such as labels or thumbnails. Each item can be "expanded" or "collapsed" to reveal the conte associated with that item.

An accordion is a UI that allows you to show multiple information panels in an orderly fashion without filling the screen with texts and other content.

Accordions should be used sparingly - if the user can't see everything they might not realize that there is more information hidden away, hence the importance of a clear icon to indicate that there is more content.
